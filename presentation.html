Présentation projet synthèse B65

MyAnimTV

Bonjour à tous, mon projet consiste à créer un site dans le but d'explorer toutes les facettes du web. Pour ce faire, je me suis inspiré des sites WEB qui diffusent des vidéos, car il y avait pleins d'intéractions entre l'interface et son utilisateur.

Voici ce que je trouve intéressant comme intéractions

  • Le Remplissage des formulaires
  • La recherche d'une vidéo avec la base de données
  • La relation de la base données avec le serveur quand on se logge
  • La gestion des comptes usagers
  • Les fonctionnalités qu'un administrateur peut faire dans son interface
  • La rentrée de plusieurs types de données (images,sons,vidéos,texte,etc)
  • La dépendance des commentaires entre chaque vidéo lié par un usager
  • La sécurité des interfaces contre le hacking
  • Le design, l'ergonomie et les effets visuels que l'on peut créer
  • Et enfin, on veut qu'un site WEB offre la meilleure expérience à son utilisateur

Pour aller dans les détails, voici quelques captures d'écrans. Vous pouvez cliquer sur l'image pour la voir en grand format

Sur cette image, on peut voir l'intéraction entre l'interface et l'utilisateur. Ici, j'ai employé une vérification des champs avec AJAX pour éviter à l'usager de rafraichir sa page lorsqu'il envoit le formulaire.

Dans la deuxième image, on voit que l'usager intéragit avec la vidéo lorsqu'il la joue. Mais surtout, il tisse un lien avec l'interface pour commenter. Il y a en arrière de lui une gestion des données entre le serveur et la base de données.

Enfin, la dernière image montre ce que l'administrateur du côté serveur puisse accomplir. On peut voir que l'admin intéragir aussi avec l'interface lorsqu'il ajoute du contenu dans la page accueil, supprime des usagers ou des vidéos, changer ses données, etc.

Les technologies utilisées pour ce projet sont :

  • Fureteur (Google chrome, FireFox, etc)
  • WAMP SERVEUR (Serveur sous Windows)
  • MYSQL (Database : base de données)
  • PHP (Langage côté serveur)
  • JAVASCRIPT (Langage côté client)
  • AJAX (Autre forme de javascript)
  • CKeditor (Librairie basée sur JQUERY)

Le temps investi pour ce projet est estimé à plus que 36 heures.
